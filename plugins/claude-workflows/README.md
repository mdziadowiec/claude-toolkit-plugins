# claude-workflows

Workflow and discipline skills for Claude Code. No external dependencies — pure prompt skills.

## Skills

- **crisp** — shorter, more direct responses; cut filler
- **karpathy** — coding discipline: think before coding, simplicity first, surgical changes, goal-driven execution
- **handoff** — produce a structured handoff note so a fresh session can continue without re-deriving context
- **doc-coauthoring** — structured workflow for co-authoring docs, proposals, specs, and decision docs

## Install

```
/plugin marketplace add mdziadowiec/claude-toolkit-plugins
/plugin install claude-workflows@claude-toolkit-plugins
```

Invoke skills as `/claude-workflows:crisp`, `/claude-workflows:karpathy`, etc.
